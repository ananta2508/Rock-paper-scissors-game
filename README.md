# 🎮 Rock Paper Scissors Game

## 📌 Overview

The **Rock Paper Scissors Game** is a Python command-line application developed as part of my **Python Programming Internship at CodSoft**. It is based on the classic hand game where the player competes against the computer. The computer makes a random choice, and the program determines the winner according to the standard game rules.

This project provides an interactive experience while demonstrating the practical implementation of Python programming concepts. It features multiple rounds, score tracking, input validation, and a user-friendly menu, making it an ideal beginner-to-intermediate Python project.

## ⚙️ How It Works

1. The program displays three choices: Rock, Paper, and Scissors.
2. The user selects an option by entering the corresponding number.
3. The computer randomly selects one of the three choices.
4. The program compares both selections using the game rules.
5. The winner of the round is displayed.
6. Scores for both the user and the computer are updated automatically.
7. The player can continue playing multiple rounds or exit the game.
8. When the game ends, the final scores and overall winner are displayed.

---

## 🎯 Game Rules

- 🪨 Rock defeats Scissors.
- ✂️ Scissors defeats Paper.
- 📄 Paper defeats Rock.
- 🤝 If both choices are the same, the round ends in a tie.
